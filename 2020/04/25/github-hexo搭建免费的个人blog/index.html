四、配置Hexo

不喜欢hexo的初始页面布局？

没关系，我们可以打开Hexo官网的主题页面，选择我们喜欢的主题

页面中有很多别人创建的主题，点击图片可以预览主题的样式，点击名字可以跳转到对应的github地址

我们可以进入选中主题的github地址，下载他的压缩包，并且解压到你创建的blog文件夹中的theme文件夹下

之后再blog文件夹下找到_config.yml文件，用文本编辑器（sublime、vscode或者其他你喜欢的编辑器）打开他，找到 theme： landscape 并将landscape改成你刚解压出来的主题的文件夹名字（注意theme:后面应该有一个空格哦）

然后在blog文件夹中右键git bash here

依次执行

再次打开http://localhost:4000就可以看到效果了

注意：有些比较炫酷的主题有着更多的配置需要去做，在选择主题的时候留意主题中是否有配置教程，根据他的教程来进行特别配置

五、配置Github

我们前面的配置，只能使我们自己在本地自娱自乐，其他人是不能在网络上搜索到我们的blog的。

所以，我们要利用Github来将自己的blog放到网络上。

github账号注册我们就不多说了，注册之后要创建一个新的仓库

仓库的名称有硬性格式要求： 你的github用户名.github.io

比如我的github用户名叫 Zhangsan，那么我的仓库名字必须是 Zhangsan.github.io
